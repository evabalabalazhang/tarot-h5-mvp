# 每日塔罗课 H5 MVP

这是第一版可点击 H5 原型，重点验证“小白每天跟着学塔罗”的核心体验。

## 怎么打开

直接打开 `index.html` 即可。

## 已包含功能

- 今日学习：首页展示第几天、今日课程、任务进度
- 第一周课程：Day 1-Day 7
- Week Progress：首页只显示当前课程和周进度，完整路径折叠展示
- 阶段地图：显示阶段一“认识塔罗”的当前位置、已完成内容和下一步
- 任务交互：课程拆成小知识块，观察和练习都需要提交后才完成
- 练习反馈：开放观察不评分，提交后展示参考观察；识别题提交后即时反馈对错
- 今日收获：可选记录，不计入学习进度
- 完成页：完成必做任务后显示课程完成反馈、连续学习天数和今日金句
- 大牌图片：已为 22 张大阿尔卡那配置 Rider-Waite-Smith 图像链接
- 课程封面：理论课显示学习插画，卡牌课支持显示对应塔罗牌
- 每日抽牌：每天抽一张牌，生成今日提醒和晚间复盘问题
- AI 问牌模拟：支持单张牌和三张牌解读
- 牌库查询：已包含 78 张维特塔罗基础牌义
- 我的成长：展示学习天数、打卡、问牌记录
