# Product Definition

## Product Name

每日塔罗课 / Daily Tarot Course

## Product Vision

A 90-day Rider-Waite tarot training system for complete beginners.

The product is designed for users who do not know tarot yet, including the creator as the first user. The goal is not to provide a card dictionary or a fortune-telling toy first. The goal is to help a complete beginner gradually build tarot thinking and basic interpretation ability.

## Core Transformation

From:

> I know nothing about tarot.

To:

> I can observe cards, understand their structure, interpret simple meanings, combine cards, and complete a beginner tarot reading on my own.

## Core Learning Loop

The product learning loop is:

> Observe → Practice → Apply → Feedback → Interpret

Every lesson should help the user interact with actual tarot cards as early as possible.

## Primary User

Complete tarot beginner.

User characteristics:

- Does not know where to start
- Does not remember card meanings
- Needs a daily plan
- Needs guided practice, not just information
- Needs confidence and feedback

## Product Principle

The product should not make the user feel:

> I read a lesson.

It should make the user feel:

> I practiced tarot today.

## 90-Day Learning Journey

### Day 1-Day 7: Tarot Thinking

Build the foundation:

- Observe cards
- Understand deck structure
- Distinguish Major and Minor Arcana
- Understand four suits
- Understand upright/reversed simply
- Ask better questions
- Complete first guided 3-card reading

### Day 8-Day 30: Major Arcana Journey

Learn and interpret all 22 Major Arcana cards through:

- Observation
- Core theme
- Application
- Interpretation
- Review
- Card relationships
- Weekly graduation exercises

### Day 31-Day 60: Minor Arcana System

Learn Minor Arcana through:

- Suit meaning
- Number meaning
- Image observation
- Real-life application

### Day 61-Day 70: Court Cards

Learn Court Cards as:

- People
- Roles
- Behaviors
- Maturity levels
- Parts of self

### Day 71-Day 85: Spread Reading

Learn to combine cards in:

- One-card readings
- Three-card spreads
- Choice readings
- Self-reflection readings

### Day 86-Day 90: Independent Graduation

User completes independent beginner readings and demonstrates basic tarot reading ability.

## Day 90 Graduation Standard

A Day 90 graduate can:

- Ask a useful tarot question
- Recognize the full deck structure
- Interpret single cards using image, type, suit, number/rank, and context
- Combine cards in simple spreads
- Complete a 3-card beginner reading
- Give grounded advice without absolute prediction
